# 다낭 여행 (Da Nang Travel) 웹사이트

베트남 다낭의 아름다움을 소개하는 상업용 관광 웹사이트입니다. 한국인 관광객을 위해 특별히 설계되었으며, 300대의 차량과 다양한 관광 서비스를 제공합니다.

![다낭 투어 웹사이트 스크린샷](screenshot.jpg)

## 특징

- 아름다운 반응형 디자인
- 한국어 및 영어 지원
- 다낭의 주요 관광지 소개
- 다양한 투어 옵션 및 서비스 제공
- 300대 차량 보유 강조
- 고급 애니메이션 및 시각 효과
- 완전 반응형 모바일 최적화 디자인
- 간편한 연락처 및 예약 시스템

## 사용 기술

- HTML5
- CSS3 (애니메이션, 그리드, 플렉스박스)
- 순수 JavaScript (프레임워크 사용 안 함)
- 폰트어썸 아이콘
- 구글 폰트

## 웹사이트 구조

```
/
├── index.html         # 메인 웹사이트 페이지
├── styles.css         # 모든 스타일 정의
├── scripts.js         # 인터랙티브 기능 및 애니메이션
├── images/            # 웹사이트에 사용된 모든 이미지
│   ├── hero-bg.jpg
│   ├── destinations/
│   ├── gallery/
│   └── fleet/
└── README.md          # 이 문서
```

## 설치 및 실행

이 프로젝트는 순수 프론트엔드 HTML/CSS/JS로 구현되었습니다. 설치가 필요 없으며 바로 브라우저에서 실행할 수 있습니다.

1. 저장소를 클론합니다:
```
git clone https://github.com/your-username/danang-travel-website.git
```

2. 브라우저에서 `index.html` 파일을 엽니다.

## 특별 기능

- 스크롤 애니메이션 - 스크롤하면서 페이지 요소들이 부드럽게 나타납니다
- 반응형 디자인 - 모든 화면 크기에 최적화되어 있습니다
- 갤러리 효과 - 다낭의 아름다운 사진들을 효과적으로 보여줍니다
- 후기 슬라이더 - 고객 후기를 자동으로 전환하는 기능
