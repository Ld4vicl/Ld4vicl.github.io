# Service Images Required

This folder should contain the following images for service icons:

1. `custom-tour.jpg` - Image representing customized tours
2. `private-transport.jpg` - Image showing private transportation services
3. `group-tour.jpg` - Image showing a group tour
4. `food-tour.jpg` - Image showing Vietnamese food or a food tour
5. `beach-activity.jpg` - Image showing beach activities in Da Nang
6. `korean-guide.jpg` - Image of a tour guide or a Korean tourist with guide

All images should be square format (same width and height) and at least 200x200 pixels in size for best display. They will be cropped to circles in the website layout.
